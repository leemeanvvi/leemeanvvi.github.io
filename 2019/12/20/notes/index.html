    deadbeef的作用

    调用栈
           +-----------------+
       |     retaddr     |
       +-----------------+
       |     saved ebp   |
ebp--->+-----------------+
       |     parameter   |
       |                 |
       |                 |
       |                 |
       |                 |
       |                 |
       +-----------------+
    构造payload
           +-----------------+
       |    system_addr  |
       +-----------------+
       |     deadbeef    |
ebp--->+-----------------+
       |  /bin/sh_addr   |
       |                 |
       |                 |
       |                 |
       |                 |
       |                 |
       +-----------------+

    payload = “A”*n + p32(system) + p32(0xdeadbeef) + p32(str_bin_sh)

    deadbeeef的作用就是覆盖saved ebp
